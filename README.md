# JavaScript TODO List
📝 Simple JavaScript To-Do List

A simple and interactive To-Do List web application built using HTML, CSS, and Vanilla JavaScript.

This project allows users to add and remove tasks using localStorage.

🚀 Features

✅ Add new tasks

🔄 Toggle task status (Done / Not Done)

❌ Remove tasks

💾 Save tasks in browser using localStorage

📱 Scrollable layout when many tasks are added

🛠️ Technologies Used

HTML5

CSS3 (Flexbox for layout)

JavaScript (DOM manipulation)

Browser localStorage for persistence
